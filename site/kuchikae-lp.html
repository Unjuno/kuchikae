What is Kuchikae?

Kuchikae is a prompt-conditioned speech transformation tool. It takes your voice input, understands the emotion and style, transforms the text according to your instructions, and generates speech that matches the desired tone — all while preserving your voice characteristics.

Voice-Conditioned

Your input audio serves as both the utterance and the voice reference. The output sounds like you, not a generic TTS voice.

Prompt-Controlled

Free-form text prompts control how your speech is transformed. Make it polite, casual, urgent, or calm — just describe it.

Emotion-Aware

Detects emotion from both text content and audio characteristics. Fusion of signals ensures natural, context-aware output.

Streaming Support

Push-to-talk mode with real-time streaming STT. Partial transcripts appear as you speak.

CLI & Web UI

Use the Gradio web interface or integrate via CLI. Perfect for automation and scripting.

Smoke Test Mode

Dummy backends let you test the full pipeline without any model downloads. Just install and run.

Quick Start

# Smoke test (no models required)
uv run python -m kuchikae.cli serve --dummy

# Real backends
uv sync --extra real
uv run python -m kuchikae.cli serve --real

# Check backends
uv run python -m kuchikae.cli doctor
