Kuchikae v0.1.0

Kuchikae

Speak once. Kuchikae says it back in your voice, following your prompt.

話した内容を、プロンプト通りに、自分の声っぽく言い直すための日本語向けプロトタイプです。

GitHub Release v0.1.0

What it does

Kuchikae records a short utterance, transforms the transcript according to a prompt, and sends the transformed text to a voice output backend. Real backend mode can use the input audio as a voice reference. Exact voice cloning is not guaranteed.

Prompt-controlled

Use free-form prompts to make speech polite, casual, urgent, calm, or formatted for a scenario.

Voice-conditioned architecture

The pipeline carries a VoiceContext so real backends can use the input audio as reference material.

Smoke test mode

Dummy backends verify the app and pipeline without downloading STT or TTS models.

Real backend mode

Optional integrations include faster-whisper, Ollama, Irodori-TTS, and experimental OpenVoice.

How it works

Record

Speak once through the Gradio UI.

Transform

Apply a prompt such as polite, movie trailer, game commentary, or space AI.

Synthesize

Generate output through dummy smoke mode or optional real voice backends.

Quick start

# Smoke test, no models required
uv sync --extra test
uv run kuchikae serve --dummy

# Real backend mode
uv sync --extra real
uv run kuchikae serve --real

# Real streaming STT
uv run kuchikae serve --real --streaming

# Diagnostics
uv run kuchikae doctor
Dummy mode is only for smoke testing. It does not evaluate real speech recognition, real voice quality, or exact voice cloning.

Status

Prototype

v0.1.0 is intended for local demos and architecture validation.

Limitations

Real quality depends on local model setup, hardware, and backend availability.
